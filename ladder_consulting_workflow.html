3. Cold Outreach Module

Email Campaign Flow

1. Upload Excel/CSV Contact List → background import → outreach_contacts
2. Create Email Campaign (select list, write template with merge tags)
3. Send Campaign (batches of 20 with 1s delay, can pause mid-send)
4. IMAP Poller (every 2 min) matches replies via tagged Reply-To address
5. Executive receives reply in Replies inbox → reply / convert to lead / ignore
6. Convert to Lead → Leads Pipeline with stage progression

Merge tags: {{first_name}} {{full_name}} {{company_name}} {{designation}} {{city}} {{executive_name}}

WhatsApp + Cold Calls

WhatsApp Campaign

  • Create template (header/body/footer + variables) → Meta approval
  • Create campaign with variable mapping → send via WhatsApp Cloud API
  • Incoming messages arrive via Meta webhook → same Replies inbox

Cold Calls

  • Dial contact, log outcome: no_answer / voicemail / callback_scheduled / interested / converted
  • Callback → notification to executive at scheduled time
  • Convert to lead directly from call log

4. Payment & Invoicing System

💳 Placement Fee

  • Triggered on exec approval of offer-letter request
  • invoice_type = placement_fee in invoices table
  • Amount = placement_fee_multiplier × offered CTC
  • Partial payments supported (amount_paid tracked)
  • PFI status mirrored via syncPlacementFeeStatus()

🎓 Training Fee

  • Triggered on admin approval of training request
  • invoice_type = training_fee in invoices table
  • Amount = price_per_user × num_users
  • Same Cashfree payment flow on /company/payments
  • Company emailed invoice number on approval

📄 HR Executive Invoices

  • Executive raises manual invoices for their companies
  • Types: placement_fee / partial_payment / other_fee / training_fee
  • Admin sees all via Payments page (Service Invoices tab)
  • Mark paid / partially paid / waived controls
